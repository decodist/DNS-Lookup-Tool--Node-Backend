## DNS Propagation Checker Tool - Node Backend

Summary:
* This tool allows anyone to check the global propagation progress of any domain name. 
* The project is in two parts - Node Backend (this repo) and a [React Frontend](https://github.com/decodist/dns-tool-react-front-end).
* The backend performs the actual DNS lookups and exposes the results via a JSON API.

### Built Using

* [Node.js](https://nodejs.org/)
* [Express.js](https://expressjs.com/)
